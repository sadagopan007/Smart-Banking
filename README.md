SmartBank — AI-Powered Banking Management System

A full-stack Flask banking app with a Random Forest fraud detection engine 
at its core. Built to demonstrate applied ML in a real-world transactional 
pipeline — feature engineering, model training, persistence, and live 
inference — alongside OOP design (encapsulation, inheritance, polymorphism) 
and a normalized MySQL schema.

🔍 ML: scikit-learn Random Forest classifier flags suspicious transactions 
   in real-time using engineered features (amount, time-of-day, transaction 
   velocity, behavioral average)
🏦 Core banking: deposits, withdrawals, transfers, interest calculation, 
   overdraft handling
🔐 Security: password-confirmed transactions, hashed credentials
📊 Live dashboard with Chart.js analytics

Tech: Python · Flask · scikit-learn · NumPy · MySQL · Bootstrap 5
